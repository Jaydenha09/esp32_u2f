# ESP32 U2F Security Key.

Turns your cheap ESP32 U2F token.

### Hardware Required

Any ESP board that have USB-OTG supported.

### Build and Flash

To build this example, set the target first:

```
# Build for ESP32-S2 for example 
idf.py set-target esp32s2
```

Launch the build:

```
idf.py build
```

Finally, flash it:

```
idf.py flash 
```
